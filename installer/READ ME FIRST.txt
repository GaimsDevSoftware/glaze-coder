glaze-coder installer

How to run it:

  1. Right-click "Install glaze-coder.command" and choose Open.
  2. In the warning that appears, click Open again.
     (This is only needed the first time. The warning shows because the
     installer is not signed with a paid Apple certificate.)
  3. A Terminal window opens and asks which parts you want. Press Enter to
     accept the suggestion in brackets, or type y or n.

What you can choose:

  Core: the glaze-dev command   (always installed)
      The engine that creates, builds and runs your apps.

  Glaze skills for Claude Code   (recommended)
      Helps Claude Code understand how Glaze apps are built.

  Claude Code plugin             (recommended)
      Adds the /glaze-coder:glaze command inside Claude Code.

  Raycast commands               (optional)
      Buttons in Raycast to make, build and edit apps without a terminal.
      Only useful if you have Raycast.

What you need first:

  - macOS (Tahoe or newer) on Apple Silicon
  - Glaze installed, with at least one app
  - Claude Code installed (for the plugin part)

More help: https://github.com/GaimsDevSoftware/glaze-coder
